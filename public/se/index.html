Bordsplaceringsgeneratorn
[GitHub]

Ett verktyg för lärare för att generera bords­placer­ingar för klasser. Verktyget är ett WPF-program skrivet i C# som använder en algoritm för att generera bordsplaceringar baserat på an­vän­darens önskemål. T.ex. att elever som inte får sitta bredvid varandra inte hamnar bredvid varandra eller att elever som behöver sitta nära tavlan hamnar nära tavlan.
